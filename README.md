# Gold Zakat Calculator

**Individual Assignment by Muhammad Aleff bin Muhammad Khairi**


## Overview
Gold Zakat Calculator is an Android application designed to help users easily calculate zakat on gold. The app allows users to input their gold holdings and current gold price to calculate the zakat amount accurately according to Islamic guidelines.

## Features
- Input gold weight in grams or tola
- Set current gold price
- Calculate zakat automatically
- User-friendly interface
